Getting Started

Install docs-server as a dev dependency:

bun add -d docs-server

Configuration

Create a docs-server.config.ts in your project root:

export default {
  title: 'My Library',
  baseUrl: '/',
  nav: ['index', 'getting-started'],
}

Writing Content

Create markdown files in docs-src/ with frontmatter:

---
title: My Page
---

# My Page Content
